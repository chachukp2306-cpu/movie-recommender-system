# Movie Recommendation System

A content-based movie recommendation engine built using Python, Machine Learning (Cosine Similarity), and Streamlit. It recommends 5 similar movies based on the user's input with real-time posters fetched via OMDb API.

## Dataset
- TMDB 5000 Movies Dataset

## How to Run Locally
1. Clone this repository.
2. Download the model files (`movie_dict.pkl` and `similarity.pkl`) from the Google Drive link provided below.
3. Place the `.pkl` files in the same project directory.
4. Run the command: `pip install streamlit pandas requests scikit-learn`
5. Run the application: `streamlit run app.py`
